# Metrics

Each section shows what the metric renders, what it needs, and what it looks like.

## `CODING_STREAK`

Streak and consistency, derived from GitHub commits. Adds time-spent rows when WakaTime is configured.

**Needs:**
- GitHub commits (always available).
- `WAKATIME_API_KEY` — optional; unlocks the time / consistency rows.

**With WakaTime:**

**📈 Coding Streak**
```
🔥 Current Streak:        14 days
🏆 Longest Streak:        45 days
📊 Daily Average:         3 hrs 44 mins
💪 Total Coding Time:     1,383 hrs 16 mins
🎯 Coding Consistency:    87.5%
📅 Active Days:           128 days
```

**Without WakaTime:**
```
🔥 Current Streak:        14 days
🏆 Longest Streak:        45 days
```

Streaks count consecutive days with at least one commit, in your `TIME_ZONE`.

## `WAKATIME_AI_STATS`

AI vs human coding attribution, aggregated over `WAKATIME_RANGE`.

**Needs:**
- `WAKATIME_API_KEY` (required). WakaTime must be tracking AI attribution (GenAI integration in your editor).
- `WAKATIME_RANGE` — same options as `WAKATIME_SPENT_TIME` (default `last_7_days`).

**🤖 My AI Footprint**
```
🤖 Generated by AI:        12.3K lines
👤 Written by Hand:        8.7K lines
📊 AI Contribution:        58.6%
🔤 Tokens In / Out:        1.2M / 3.4M
💬 Total Prompt Chars:     484.8K
```

The block hides itself entirely when no AI activity is reported. The prompt row reads `Total Prompt Chars` (raw `ai_prompt_length`) today; once WakaTime exposes `ai_average_prompt_length` at top level the row becomes `Average Prompt`.

The title adapts to the range:

| `WAKATIME_RANGE`     | Title                    |
|----------------------|--------------------------|
| `last_7_days`        | **🤖 My Week in AI**     |
| `last_30_days`       | **🤖 My Month in AI**    |
| `last_6_months`      | **🤖 My 6 Months in AI** |
| `last_year`          | **🤖 My Year in AI**     |
| `all_time` (default) | **🤖 My AI Footprint**   |
